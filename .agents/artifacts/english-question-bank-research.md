## Key Insights

### Common Question Formats
1. **Multiple Choice** (most common)
   - 4 options (A, B, C, D)
   - Clear question stem
   - One correct answer
   - Explanation provided

2. **True/False**
   - Statement-based
   - Binary choice
   - Used for comprehension checks

3. **Fill in the Blanks**
   - Complete sentences
   - Test grammar/vocabulary in context
   - Multiple blanks per question

4. **Gap Filling**
   - Similar to fill-in-blanks
   - Focus on articles, prepositions, tenses

### Question Quality Standards

Based on institution standards:

✅ **Each Question Must Have:**
1. Clear, grammatically correct question
2. 4 options (one correct, three distractors)
3. Correct answer index (0-3)
4. Detailed explanation (2-3 sentences)
5. Difficulty level (easy/medium/hard)
6. Topic/category tag
7. CEFR level (A1, A2, B1, B2, C1)

✅ **Explanation Format:**
```
"The correct answer is 'goes' because with third-person singular subjects 
(he, she, it), we add -s or -es to the base verb in present simple tense. 
'She goes to school' follows this rule."
```

✅ **Distractor Quality:**
- Must be plausible but incorrect
- Should test common mistakes
- Include typical errors (e.g., "she go" for third person)
