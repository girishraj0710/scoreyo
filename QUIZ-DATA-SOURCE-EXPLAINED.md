## 🔍 Where Quiz Data Comes From

Krakkify uses a **3-tier hierarchical system** to generate quizzes:

```
┌─────────────────────────────────────────────────────────────┐
│                    QUIZ GENERATION FLOW                     │
└─────────────────────────────────────────────────────────────┘

User Requests Quiz (Exam, Subject, Topic, Difficulty, Count)
                          ↓
        ┌─────────────────────────────────────┐
        │  TIER 1: Verified Question Bank    │
        │  (exam_questions table)              │
        │  • Human-reviewed questions          │
        │  • Instant delivery (0 latency)      │
        │  • Highest quality                   │
        │  • source: "verified"                │
        └─────────────────────────────────────┘
                          ↓
                   Not enough? → Try Tier 2
                          ↓
        ┌─────────────────────────────────────┐
        │  TIER 2: Cached AI Questions        │
        │  (cached_questions table)            │
        │  • Previously generated by AI        │
        │  • Instant delivery (0 latency)      │
        │  • Good quality (deduplicated)       │
        │  • source: "ai"                      │
        └─────────────────────────────────────┘
                          ↓
                   Still not enough? → Try Tier 3
                          ↓
        ┌─────────────────────────────────────┐
        │  TIER 3: Fresh AI Generation        │
        │  (OpenRouter API)                    │
        │  • Real-time AI generation           │
        │  • 12-14 second timeout              │
        │  • Variable quality                  │
        │  • source: "ai"                      │
        │  • Auto-saved to cache for Tier 2    │
        └─────────────────────────────────────┘
                          ↓
            ✅ Quiz Generated & Delivered
```

## 🐛 What Caused the "AI Warming Up" Error?

### **Error Message**:
```
Our question generator is a bit slow for Semiconductors & Communication 
(Diodes, Transistors, Logic gates) right now, and this topic isn't cached yet.

Try again in a few seconds, or pick a different topic — popular ones are 
usually instant.
```

### **Root Cause Analysis**:

When you requested a quiz for **NEET Physics → Semiconductors & Communication**:

**Tier 1 (Verified Bank)**: 
- Checked `exam_questions` table
- Found: **0 questions** ❌
- Reason: Topic was never seeded

**Tier 2 (Cached Questions)**:
- Checked `cached_questions` table
- Found: **0 questions** ❌
- Reason: No previous user had generated this quiz, so cache was empty

**Tier 3 (AI Generation)**:
- Called OpenRouter API with 14-second timeout
- Result: **Failed or timed out** ❌
- Reasons could include:
  - Model API slow/overloaded
  - Network latency
  - Rate limiting
  - Complex topic requiring longer generation time

**Outcome**: All 3 tiers failed → Error shown to user

## 🎯 Best Practices

### **For Popular Topics**:
- Seed at least **20-50 verified questions**
- Mix difficulty levels (30% easy, 50% medium, 20% hard)
- Ensure high-quality explanations
- Mark as `source: "verified"`

### **For Niche Topics**:
- Seed at least **10-15 questions** minimum
- AI generation will supplement if needed
- Cache will build up over time as users take quizzes

### **Quality Checklist**:
- [ ] Accurate correct answer
- [ ] Clear question statement
- [ ] 4 distinct options
- [ ] Detailed explanation (why correct, why others wrong)
- [ ] Appropriate difficulty level
- [ ] No duplicates in bank

## ❓ Troubleshooting

### **Error: "AI warming up"**

**Diagnosis**:
```bash
# Check Tier 1
SELECT COUNT(*) FROM exam_questions 
WHERE exam_id = ? AND topic = ?;

# Check Tier 2
SELECT COUNT(*) FROM cached_questions 
WHERE exam_id = ? AND topic = ?;

# If both are 0, seed questions!
```

**Solution**: Seed 10-20 verified questions for the topic

---

### **Error: "Quiz generation timeout"**

**Diagnosis**: AI generation (Tier 3) is timing out

**Causes**:
- OpenRouter API overloaded
- Model rate limits hit
- Network issues
- Complex topic requiring long generation time

**Solutions**:
1. **Immediate**: Seed verified questions (Tier 1)
2. **Short-term**: Wait for cache to warm (Tier 2)
3. **Long-term**: Increase timeout or use faster model

---

### **Error: "Daily quiz limit reached"**

**Diagnosis**: Free user hit 3 quizzes/day limit

**Solution**: Upgrade to Pro ($79/month) for unlimited quizzes
